## 🚀 스터디 진행 방식

1.  **주간 진행 상황 공유**: 매주 화요일, 각자 한 주간의 진행 상황과 목표 달성 여부를 공유합니다.
2.  **자율적 코드 관리**: 개인 폴더에 자신의 소스 코드를 자유롭게 업로드하여 관리합니다.
3.  **지식 공유 세션**: 스터디 그룹 내에서 누구나 자율적으로 세션을 열어 특정 주제에 대한 지식을 공유하거나, 다 함께 모여 코딩하는 시간을 가질 수 있습니다.
